A simple plugin to add a positive/negative user review page and widget to your website.

== Description ==

Simple Website Review allows you to add a review page to your website linking to external review websites or allowing your users to fill in an email form depending on the review type.
Simple Website Review also includes a customisable widget which can link to your review page.

== Installation ==

1. Upload `simple-website-review` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
